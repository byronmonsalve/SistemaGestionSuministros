## Descripción

Microservicio desarrollado para el proyecto **Sistema de Control de Suministros Tecnológicos (CloudTech)** de la asignatura Desarrollo FullStack I.

Este microservicio administra la información de órdenes de compra en tránsito y responde las consultas realizadas por **api.suministros** cuando un componente presenta stock crítico.
