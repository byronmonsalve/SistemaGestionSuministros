## Descripción

Microservicio desarrollado para el proyecto **Sistema de Control de Suministros Tecnológicos (CloudTech)** de la asignatura Desarrollo FullStack I.

Este microservicio administra el inventario de componentes tecnológicos, permitiendo registrar, consultar, actualizar y eliminar suministros. Además, cuando el stock de un componente es menor a tres unidades, realiza una consulta al microservicio **api.proveedores** para verificar si existen órdenes de compra en tránsito.
